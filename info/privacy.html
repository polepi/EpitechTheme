Personally Identifiable Information:
EpiTheme, a productivity extension tailored for Epitech students, collects certain personally identifiable information to enhance user experience. Specifically, the extension requires access to the user's Epitech API userid, email, and name. This information is essential for accessing the Epitech API functionalities within the extension. Rest assured, all collected data remains securely stored within the localStorage of your device and is not shared with any third parties.

Authentication Information:
To ensure seamless integration with Epitech and optionally Trello APIs, the extension necessitates the provision of API tokens. These tokens are used exclusively for authenticating and authorizing access to the respective APIs. While the Epitech API token is mandatory, providing the Trello API token is at the user's discretion. Similar to personally identifiable information, all authentication data, including API tokens, is securely stored within the extension's localStorage and is not disclosed to external entities.

Data Storage and Sharing:
Any information collected or generated by EpiTheme is confined solely within the localStorage of the extension. We uphold strict confidentiality standards, ensuring that user data is not shared, transmitted, or disclosed outside the confines of the user's device. Your privacy and security are paramount, and we are committed to maintaining the integrity and confidentiality of your information.

Open Source Code:
We believe in transparency and collaboration. The source code for EpiTheme is open source and accessible at our github. Feel free to review, contribute, or raise any concerns regarding the codebase (github private until the extension has been released).
