Install themes

1Download the Theme folder. (with theme.css file inside)

📁 Theme/
    📃 theme.css
    📃 readme.txt (optional)

2Drag the folder inside the extension folder

🗃️ For unpacked extensions (github extension)

(Extension Folder)/Themes

📦 For packed extensions (google extension marketplace)

🟦 Windows

C:\Users\(your username)\AppData\Local\Google\Chrome\User Data\Default\Extensions\\Themes

🐧 Linux

~/.config/chromium/Default/Extensions/(extension id)/Themes

🍎 macOS

Use a normal operational system, ty

Create themes

1Rename your CSS file to theme.css

2Put your CSS file inside an empty folder, and rename the folder as you wish to call the Theme

3Follow the same steps as "Install themes"

Tip:You can copy the Dark or Light theme files and use it as a base for your own Theme.
