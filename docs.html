Overview

1Local list

Local lists contain your tasks, and are stored in your local device.

Create list

1. Press menu

2. Assign a new to the list. eg "my_ls"

Open list

1. Press menu

2. Click on your desired list

Create card

1. Press add

2. Enter the required information and click "Create"

Create card (Intranet)

1. From the intranet, open a project page

2. Press playlist_add

A new card has been added to your currently selected list with the information of the project automatically

2Trello list

Requires Trello API connection (Docs)

Links a list from Trello

Link a list

1. Press menu, "Trello" (top right-hand)

2. Select your Trello list from the list, or input your Trello List ID in the textbox (and press Enter)
